An analogy to describe the differences between HTML and CSS

Html and CSS with JavaScript, all work together to create dynamic and interactive websites or web applications.
Html is responsible for the content of the page, or structure of it.
The text, images, buttons, title, and headings. It also links to different
webpages inside and out. On the other hand, CSS is responsible for the presentation of the content.
Basically for styling, like Colors, font, positioning of elements.
We can think Html like a builder and CSS as the artist.

the same webpage with Html only.
